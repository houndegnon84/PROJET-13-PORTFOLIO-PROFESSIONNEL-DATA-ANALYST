# Portfolio Data Analyst – Projet Aéroworld

Portfolio orienté vers la candidature au poste de Data Analyst, Chef de projet data analyst pour Aéroworld, une entreprise de conception, développement, fabrication et maintenance d'avions. Le portfolio comprend :

- L'analyse des besoins métier qui a permis d'identifier les besoins stratégiques de Aéroworld;
- Le cahier des charges fonctionnel pour cadrer les besoins et les solutions;
- Un diagramme de Gantt, outil de gestion de projet présentant le planning d'élaboration du portfolio;
- Un Dashboard Profil présentant mon parcours, mes compétences ainsi que mes projets data;
- Un Dashboard Veille stratégique orienté sur les outils et technologies actuelles data;
- Un tutoriel de formation sur la création de graphique dans Power BI (vidéo + documentation)
